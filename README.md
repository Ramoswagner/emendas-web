# Gestão de Emendas — Hospital da Baleia

Sistema web de gestão de emendas parlamentares do Núcleo de Projetos.

## Acesso
[Entrar no sistema](https://seuusuario.github.io/emendas-web/login.html)

## Módulos
- **RG** — Cadastro e gestão de emendas
- **Compras** — Ordens de compra por categoria
- **Financeiro** — Notas fiscais e pagamentos
- **Contabilidade** — Prestação de contas
- **BI** — Dashboard e indicadores

## Tecnologia
- Frontend: HTML/CSS/JS (GitHub Pages)
- Backend: Google Apps Script
- Banco de dados: Google Sheets
